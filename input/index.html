Output

In the previous lesson, you wrote your first program. It made the computer spit out the text "hello world!" But what did you actually do? Well, you told the computer to write a string (more on that later) to the console. You do this by using the print command. Try it out yourself! Change the text between the quotes to whatever you want, and the computer will spit that out! AS we learn more, this command will become even more useful.



Input

There are two commands to take input in python 2.7, one of which you should never use. This is removed and the other one is renamed in python 3. To take input, use the following code. Remember, python 2 on the left, python 3 on the right.

raw_input() input()

Now run your program. You should notice it doesn't do anything. Type anything you want in, and hit return or enter. You should notice that still, nothing happened. This is because we are not actually processing the input. For now, lets just assign it to a variable. Do:

input = raw_input() input = input()

You should notice that, after you run your program, nothing changed. This is because we still aren't doing anything with the input. Also, note that you can prompt the user for an input by putting a string(text inside quotation marks) inside the brackets. See examples below.



Putting it together

Now, you know how to take input and write output. Now, what if you want to do something with your input? Well, you have to use your print command from before! Try this sample code to say hello to someone.

name = raw_input("What is your name? ")
print "Hello, " + name name = input("What is your name? ")
print("Hello, " + name)
