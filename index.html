What is Python?

Python is a weakly typed programming language. This means that you do not have to declare types for integers.

For example, in a strongly typed language, for example C#, to make an integer as 5, you would have to do:

int x = 5;

While in Python you would do:

x = 5

Notice the absence of a semi-colon. These are not required in Python.

How do I install Python?

Installing Python is very simple. Go to python.org and download either the latest version (3.6.1 at the time of writing) or download version 2.7.13. I will be covering both versions.
Now you need a decent Python editor. Your Python installation comes with one, but it isn't very good. I recommend Pycharm, however you are free to use whatever you like.
