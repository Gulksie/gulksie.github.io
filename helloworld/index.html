Your First Python Program

Ok, so now we are going to make your first python program. Start by making a new folder for all your Python projects. Inside this folder make a new file and call it "helloworld.py". Open this with your Python editor.
For reference, Python 2.7 will be on the left and Python 3.6 will be on the right.

Now, write the following text into your code

print "hello world!" print("hello world!")

Congrats! You have just written your first program! In Pycharm, press Alt+Shift+F10 to run the program. In the default interpreter, press F5 to run.
You should get the following output:
hello world!
If you didn't get that output, double check your code and make sure it is exactly the same as what is above.
